# 📚 Book Tinder — Мобильное приложение для любителей чтения

[![Kotlin](https://img.shields.io/badge/Kotlin-2.3.0-purple.svg)](https://kotlinlang.org/)
[![Android](https://img.shields.io/badge/Platform-Android-green.svg)](https://developer.android.com)
[![MAI](https://img.shields.io/badge/MAI-Project-blue.svg)](https://mai.ru)

**Book Tinder** — это инновационное решение для активных читателей, которое превращает выбор книги и поиск единомышленников в быстрый и увлекательный процесс. Мы объединили механику свайпов с глубоким социальным взаимодействием.

---

## 💡 Основная идея
Приложение является фундаментальным упрощением двух ресурсозатратных процессов:
1. **Выбор следующей книги:** Вместо бесконечного скроллинга списков — быстрая флип-лента с книгами, подобранными по вашим интересам.
2. **Поиск собеседника:** Возможность мгновенно написать рецензию на прочитанную книгу и обсудить её с другими пользователями.

> **Jobs-To-Be-Done:** «Когда я завершаю читать книгу, я хочу сразу ее обсудить с кем-нибудь, уже прочитавшим ее, чтобы разобрать важные сюжетные повороты».

---

## 🛠 Функциональные возможности

### Основные разделы:
* **🔥 Лента:** Флип-карточки для быстрого подбора. Подгрузка названий, авторов и обложек через внешнее API.
* **🔍 Каталог:** Система фильтров, поиск и рекомендации на основе предпочтений.
* **⭐ Избранное:** Удобное управление списками («Хочу прочитать», «Читаю», «Прочитано»).
* **💬 Рецензии:** Создание и просмотр рецензий, возможность обсуждения книг с другими пользователями.
* **👤 Профиль:** Персонализация данных, история просмотров и управление подпиской.
